## 维护约定

- `README.md` 只负责说明仓库定位和文档导航。
- 协作方法正文放在独立主题文档里，不要把所有经验堆在首页。
- 每一篇主题文档尽量聚焦一个主题，方便后续持续补充和复用。
- 从项目仓库同步过来的经验文档，建议保留来源背景、适用范围，必要时补一句脱敏说明。
- 如果文档值得长期复用，优先写成中文，降低传播和复用门槛。
